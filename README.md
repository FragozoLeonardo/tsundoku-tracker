## 🇺🇸 English
**Tsundoku Tracker** is a specialized management tool designed to help readers organize their unread book collections. This project serves as a technical showcase for my **MBA in Ruby on Rails**, implementing a scalable architecture focused on international markets.

* **Stack**: Rails 8 (Solid Stack), PostgreSQL, Hotwire.
* **Core Goal**: Apply clean code principles, TDD, and i18n to a real-world problem.

---

## 🇧🇷 Português
O **Tsundoku Tracker** é uma plataforma para organizar e dar vazão às pilhas de livros acumuladas. Desenvolvido como projeto prático do meu **MBA em Ruby on Rails**, ele foca em padrões de projeto de nível pleno e escalabilidade global.

* **Stack**: Rails 8, PostgreSQL, Tailwind CSS e Hotwire.
* **Destaques**: Uso de Service Objects, cobertura de testes com RSpec e internacionalização completa.
